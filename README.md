# Budget JENIE - Budget Planner

A simple web app to track your income and expenses with a light/dark mode and easy navigation.

# Features
- Login page with any username/password allowed
- Add income and expense entries
- View all records
- Light and dark theme toggle with saved preference
- Responsive and clean design

# How to Use
1. Open `login.html` and login with any credentials.
2. Navigate through the app using buttons.
3. Add your income or expenses.
4. View your records.

# Technologies Used
- HTML
- CSS
- JavaScript

# How to Run
Open the project folder in VS Code and use the Live Server extension, or open `index.html` in your browser directly.
